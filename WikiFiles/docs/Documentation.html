The extended system namespace with the GuidEx Guid extention.

 

Represents a globally unique identifier (GUID). Can create version 5 GUID.

 

Syntax

public struct GuidEx : IComparable

  

Remarks

A GUID is a 128-bit integer (16 bytes) that can be used across all computers and networks wherever a unique identifier is required. Such an identifier has a very low probability of being duplicated. The version 5 creation is built uppon the RFC4122 GUID definition.
